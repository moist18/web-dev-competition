# 🌱 EatWise2 - Complete Feature List

## ✨ **Account Settings - Full Features**

### 📱 **Access Points**
- From User Profile → "Edit Profile" button
- From User Profile → "Notifications" button  
- From User Profile → "Privacy & Security" button
- Direct URL: `/account-settings`

### 🎯 **4 Main Tabs**

#### 1️⃣ **Profile Tab** 👤
- ✅ **Profile Photo Management**
  - Upload new profile picture
  - Camera button for quick access
  - Image preview with gradient background
  
- ✅ **Basic Information**
  - Edit full name (with User icon)
  - Update email address (with Mail icon)
  - Real-time input validation
  - Save changes with animated button

#### 2️⃣ **Health Tab** 🎯
- ✅ **Health Metrics**
  - Age input (years) with Calendar icon
  - Height input (cm) with Ruler icon
  - Current weight (kg) with Weight icon
  - Target weight (kg) with Target icon
  - Each field has custom color coding (blue, green, orange, purple)

- ✅ **Daily Calorie Target**
  - Interactive range slider (1000-4000 kcal)
  - Real-time value display
  - Step increments of 100 kcal
  - Gradient visual feedback

- ✅ **BMI Calculator**
  - Automatic BMI calculation from height & weight
  - Color-coded indicator card
  - Progress bar visualization
  - Category display (Normal Range, etc.)

#### 3️⃣ **Preferences Tab** ⚙️
- ✅ **Notification Settings**
  - 🍽️ Meal Reminders - Get notified when it's time to eat
  - 📊 Daily Reports - Daily summary of your nutrition
  - 🏆 Achievements - Celebrate your milestones
  - 📢 Marketing - Updates and promotional offers
  - Toggle switches with smooth animations
  - Individual on/off controls

- ✅ **Language Selection**
  - 🇮🇩 Bahasa Indonesia
  - 🇬🇧 English
  - Flag emoji display
  - Button selection with gradient highlight

- ✅ **Theme Selection**
  - ☀️ Light Mode
  - 🌙 Dark Mode
  - Visual theme previews
  - Gradient selection buttons

#### 4️⃣ **Security Tab** 🔒
- ✅ **Change Password**
  - Current password input with show/hide toggle
  - New password input with show/hide toggle
  - Confirm password validation
  - Minimum 6 characters requirement
  - Password mismatch detection
  - Success confirmation message
  - Expandable/collapsible form

- ✅ **Security Tips Section**
  - Password strength guidelines
  - Security best practices
  - Checklist format with tips

- ✅ **Danger Zone**
  - Delete account option
  - Warning message
  - Confirmation dialog
  - Red gradient alert styling

### 🎨 **UI/UX Features**
- ✅ **Smooth Animations**
  - Tab transitions with Motion/React
  - Staggered content loading
  - Hover effects on all interactive elements
